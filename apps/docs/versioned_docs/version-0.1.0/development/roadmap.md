# Roadmap

Aura is under active development. This page describes **direction** at a high level only.

Detailed engineering plans live in the repository **[`docs/`](https://github.com/joylitix/aura/tree/main/docs)** (for example **[`docs/mvp.plan.md`](https://github.com/joylitix/aura/blob/main/docs/mvp.plan.md)**), not in these published user docs. See [CONTRIBUTING](https://github.com/joylitix/aura/blob/main/CONTRIBUTING.md) for workflow. **MVP track:** GitHub [milestone MVP](https://github.com/joylitix/aura/milestone/1) and label [`mvp`](https://github.com/joylitix/aura/issues?q=is%3Aissue+label%3Amvp).

## Direction

- **Near term**: tighten the core loop—VS Code experience, a local agent process, and safe-by-default tooling for grounded answers.
- **Next**: expand modes and integrations, improve self-hosting and operations documentation, and ship features as they become stable enough for end users.

As capabilities mature, user-facing guides and reference pages will appear in this documentation site.
