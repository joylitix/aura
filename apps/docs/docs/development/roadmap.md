# Roadmap

Aura is under active development. This page describes **direction** at a high level only.

Detailed engineering and delivery plans live **in the repository** (for contributors with repo access), not in the published documentation site. See [CONTRIBUTING](https://github.com/jennifer-dickinson/aura/blob/main/CONTRIBUTING.md) for how to work on the project and where internal planning material is kept.

## Direction

- **Near term**: tighten the core loop—VS Code experience, a local agent process, and safe-by-default tooling for grounded answers.
- **Next**: expand modes and integrations, improve self-hosting and operations documentation, and ship features as they become stable enough for end users.

As capabilities mature, user-facing guides and reference pages will appear in this documentation site.
