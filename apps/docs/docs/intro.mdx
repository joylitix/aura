# Aura documentation

**Aura** (Joylitix) is a self-hosted agent platform: VS Code extension, daemon, CLI, and (later) a desktop client—**Plan · Brainstorm · Ask · Agent · Debug** with an MCP-friendly tool layer.

Use the sidebar to browse. **Development → Roadmap** summarizes public direction only.

Public docs will grow here as features ship; see [CONTRIBUTING](https://github.com/jennifer-dickinson/aura/blob/main/CONTRIBUTING.md) for how we write and release documentation (and where contributor planning lives).
